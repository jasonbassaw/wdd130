Home page

The home page will be the page holding the other pages. It will be like the primary point that will serve as a hub linking to the various site or pages of my portfolio online site. it will come with a portfolio logo and a hero banner that will quickly get the attention of visitors. These features of visuals are to give visitors an immersive experience to stay in mi site to get to know me more. It will have the navigation bar which will contain and guide visitors to the links of my About Me page and my Work Experience page. It will also contain the Contact Me which will hold my primarily contact details such as my phone number, email address etc. It will also have the blog link also. It is going to give a brief statement that will summarize my background and what is there to be seen on the page. It will go ahead and include the footer which will establish a strong online presence by showcasing my social media platforms, such as Instagram, LinkedIn and Facebook. These platforms will allow visitors to easily connect with me, getting to exploring my personality, my interest, skills, educational background as well as professional journey etc.

Dynamic Elements and JavaScript Functionality

My portfolio website will incorporate multiple dynamic features powered by JavaScript to enhance interactivity and user experience. These features demonstrate my understanding and use of modern JavaScript techniques, including JS Objects, ES Modules, Arrays, and array methods.

Quote Slideshow

A rotating quote slideshow on the homepage uses a JavaScript array containing motivational quotes. The setInterval() method updates the quote every 5 seconds. This feature utilizes DOM manipulation and demonstrates working with arrays and dynamic content updates.

About Me Slideshow

On the About Me page, I implemented an image slideshow that cycles through pictures of me. This is built using a collection of image elements accessed with document.querySelectorAll() and uses JavaScript logic to rotate visibility using array-like structures and looping.

JS Objects

To dynamically generate content, I will use JavaScript objects to store data such as my personal profile and skills. This allows for flexible rendering of content using object properties. For example, a userProfile object can be used to insert my name, degree, and school into the DOM dynamically.
